# Services

All services running in the cluster, grouped by namespace.

| Namespace | Description |
|-----------|-------------|
| [addons](addons.md) | Cluster infrastructure — ArgoCD, cert-manager, MetalLB, Longhorn |
| [automation](automation.md) | Home automation — Home Assistant, ESPHome, Zigbee2MQTT, Node-RED |
| [databases](databases.md) | Shared databases — PostgreSQL, MariaDB, Redis, InfluxDB |
| [homelab](homelab.md) | Self-hosted apps — Gitea, Mealie, Ghostfolio, Firefly, and more |
| [ingress](ingress.md) | Access & security — Authentik, ingress-nginx, Cloudflared, Wireguard |
| [monitoring](monitoring.md) | Observability — Grafana, Prometheus, Loki, Alloy, Uptime Kuma |
